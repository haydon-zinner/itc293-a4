React Dashboard
(work in progress)

The dashboard app is a full stack project I have been working on that serves as a comprehensive digital newspaper, utilizing multiple APIs to provide me with relevant information.

This React application leverages Redux to efficiently manage the state of API calls, eliminating the need for repetitive invocations.

API's used:
  • Numbers API
  • News API
  • Weather API
  • Exchange Rates API

This application is developed as a full stack app, encompassing a Node server powered by Express to handle API calls. The client interacts with these endpoints, receiving the necessary data to render the components seamlessly.

Other features include state management for Light or Dark mode as well as a draw side menu.

Note: OnRender.com is a free hosting service which suspends webservices after inactivity. Your first visit will prompt the webservice to start up again, and then will run fine.
